# 🐐 Screaming Goat - Landing Page de Alfombras Tufting

## 1. 📌 Descripción

Landing page desarrollada como proyecto educativo de frontend.
Simula el sitio web de un emprendimiento de alfombras hechas a mano con técnica tufting, incluyendo información del proceso, trabajos realizados y contacto.
